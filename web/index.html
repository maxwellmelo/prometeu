Open source LLM infrastructure

Shared inference capacity for communities.

Prometeu turns ordinary machines into a distributed network for serving open source language models. It starts with Q8 bootstrap models for quality, then scales toward medium and large models as community capacity grows. Users can consume capacity, contributors can run nodes, and the network measures real work through signed receipts and reciprocity.

What Prometeu does

/v1 inference gateway

Apps, CLIs, bots and UIs call a common chat/completions endpoint with streaming support.

Model pool orchestration

The coordinator sizes GGUF models, selects peers, warms pools and tracks READY/DEGRADED/FAILED state.

Participant nodes

Anyone can contribute controlled CPU/RAM/GPU capacity through a local daemon and dashboard.

Hash-pinned allowlist

Nodes only load curated models with known sha256 hashes. No unverified fallback.

Reciprocity ledger

Serving tokens earns standing. Contributors receive more usage headroom while anonymous users keep a baseline floor.

Open observability

Pools, peers, registry state and Prometheus metrics expose what the network is doing.

How it works

App/User ──POST /v1/chat/completions──▶ Gateway
                                            │
                                            ├─ rate limit + token metering
                                            ├─ model/pool routing
                                            ├─ reciprocity accounting
                                            ▼
                                 Pool / participant node
                                            │
                                            ├─ verify allowlisted model hash
                                            ├─ run sandboxed inference
                                            └─ emit signed receipts

Start building

Use the API

curl -X POST http://YOUR_COORDINATOR:3000/v1/chat/completions \
  -H 'content-type: application/json' \
  -d '{"model":"qwen","messages":[{"role":"user","content":"Hello"}]}'

Run a node

curl -fsSL https://raw.githubusercontent.com/maxwellmelo/prometeu/main/node/install.sh | sudo bash
