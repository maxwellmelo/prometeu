Run a Prometeu node

You do not need to be a programmer. A node is a machine you allow Prometeu to use within limits you control. The installer sets up a local dashboard, detects your hardware, starts a service, and reports health to the coordinator.

Fast install

Recommended for Debian/Ubuntu machines.

curl -fsSL https://raw.githubusercontent.com/maxwellmelo/prometeu/main/node/install.sh | sudo bash

What you control

  • CPU limit
  • RAM limit
  • participation mode
  • models allowed by catalog
  • when to stop the service

Guided steps

1

Prepare a Linux machine

Use Debian/Ubuntu if possible. Keep it plugged in and connected to internet.

2

Run one command

Copy install command above into a terminal. It installs packages, node daemon, local dashboard and inference binaries when available.

3

Open local dashboard

After install, open http://localhost:8787 on that machine to view hardware, limits and preflight checks.

4

Watch public dashboard

Your node should appear in network dashboard if it can reach coordinator.

Useful commands

Check service

systemctl status prometeu-node --no-pager

View logs

journalctl -u prometeu-node -f

Preflight

curl http://localhost:8787/api/node/preflight

Stop node

sudo systemctl stop prometeu-node
