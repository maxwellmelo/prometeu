Run a Prometeu node

You don't need to be a programmer. A node is a machine you allow Prometeu to use within limits you control. The installer sets up a local dashboard, detects your hardware, starts a service, and reports health to the coordinator.

Fast install

Recommended for Debian / Ubuntu machines.

curl -fsSL https://raw.githubusercontent.com/maxwellmelo/prometeu/main/node/install.sh | sudo bash

What you control

  • CPU limit
  • RAM limit
  • Participation mode
  • Models allowed by catalog
  • When to stop the service

Guided steps

Four steps from a bare Linux machine to a node visible on the public network.

1

Prepare a Linux machine

Use Debian / Ubuntu if possible. Keep it plugged in and connected to the internet.

2

Run one command

Paste the install command above into a terminal. It installs packages, the node daemon, a local dashboard and inference binaries when available.

3

Open the local dashboard

After install, open http://localhost:8787 on that machine to view hardware, limits and preflight checks.

4

Watch the public dashboard

Your node should appear on the network dashboard once it can reach the coordinator.

Useful commands

Manage the node service after installation.

Check service

systemctl status prometeu-node --no-pager

View logs

journalctl -u prometeu-node -f

Preflight

curl http://localhost:8787/api/node/preflight

Stop node

sudo systemctl stop prometeu-node
